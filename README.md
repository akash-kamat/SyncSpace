# Canvas Connect

A collaborative whiteboard application built with React, Supabase, and Tldraw.

## Features

- **Real-time Collaboration**: Draw and brainstorm with others in real-time.
- **AI Analysis**: Analyze your whiteboard content using Gemini AI.
- **Secure**: Built with Supabase Auth and RLS.
- **Responsive**: Works on all devices.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Shadcn UI
- **Whiteboard**: Tldraw
- **Backend**: Supabase (Database, Auth, Realtime, Edge Functions)
- **AI**: Google Gemini API

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up Supabase project and environment variables.
4.  Run development server: `npm run dev`
